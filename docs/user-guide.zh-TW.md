# NIC-Scheduler 使用手冊

## 登入系統
- 預設管理員：admin / admin123
- 首次登入後請立即修改密碼

## 單次排程
1. 選擇網路卡
2. 設定日期 (YYYY/MM/DD)
3. 設定連接與斷開時間
4. 點擊「Add to List」

## 週期性排程
1. 選擇網路卡
2. 勾選需要排程的星期幾
3. 設定連接與斷開時間
4. 點擊「Add Recurring Schedule」

## 管理功能
- 刪除排程：選中排程後點擊「Delete Selected」
- 測試網路卡：點擊「Test NIC Operations」
- 查看日誌：切換到「Logs」頁籤
- 管理使用者：管理員可在「Administration」頁籤管理使用者
